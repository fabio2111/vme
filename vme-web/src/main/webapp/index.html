Welcome to VME RESTful web service !!! 



If you want to see the documentation of this service ... http://km.fao.org/FIGISwiki/index.php/VME_Search_Rest_Service
If you want to get the code list of the authorities in json format http://figisapps.fao.org/figis/ws/vme/webservice/references/authority/en/list
If you want to get the code list of VME types in json format http://figisapps.fao.org/figis/ws/vme/webservice/references/type/en/list
If you want to get the code list of VME criterias in json format http://figisapps.fao.org/figis/ws/vme/webservice/references/criteria/en/list
If you want to get the data related years in json format http://figisapps.fao.org/figis/ws/vme/webservice/references/years/en/list
To having an example of  how...
...query the data (json format service): http://figisapps.fao.org/figis/ws/vme/webservice/search?authority=20010&vme_type=10&year=2012
...to get a single vme item by its id  (unspecified year, it assumes current one) http://figisapps.fao.org/figis/ws/vme/webservice/get?id=10
... to get a single vme item by its id (year specified) http://figisapps.fao.org/figis/ws/vme/webservice/get?id=10&year=2010
...to get a single vme item by its inventory identifier (unspecified year, it assumes current one) http://figisapps.fao.org/figis/ws/vme/webservice/get?inventoryIdentifier=VME_NAFO_12
...to get a single vme item by its inventory identifier (year specified) http://figisapps.fao.org/figis/ws/vme/webservice/get?inventoryIdentifier=VME_NAFO_12&year=2010
...to get a single vme item by its geographic feature ident.(unspecified year, it assumes current one) http://figisapps.fao.org/figis/ws/vme/webservice/get?geographicFeatureId=VME_NAFO_10_2009
...to get a single vme item by its geographic feature identifier (year specified) http://figisapps.fao.org/figis/ws/vme/webservice/get?geographicFeatureId=VME_NAFO_10_2009&year=2012
